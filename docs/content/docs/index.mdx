# Welcome to Trino

Trino is a fast distributed SQL query engine for big data analytics that helps you explore your data universe.

## What is Trino?

Trino is an open-source distributed SQL query engine designed to query large data sets distributed over one or more heterogeneous data sources.

### Key Features

- **Fast**: Designed for low-latency queries
- **Scalable**: Can query data where it lives without moving it
- **Versatile**: Supports querying data from multiple sources
- **Open Source**: Apache 2.0 licensed
