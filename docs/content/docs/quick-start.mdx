# Quick Start Guide

This guide will help you get started with Trino quickly.

## Prerequisites

Before you begin, ensure you have:

- Java 21 or higher
- At least 4GB of available memory

## Installation

### Download Trino

```bash
curl -O https://repo1.maven.org/maven2/io/trino/trino-server/latest/trino-server-latest.tar.gz
tar xzf trino-server-latest.tar.gz
cd trino-server-*
```

### Configure Trino

Create a configuration directory:

```bash
mkdir etc
```

Create basic configuration files in the `etc` directory.

### Start Trino

```bash
bin/launcher start
```

## Running Queries

Once Trino is running, you can connect to it using the Trino CLI:

```bash
bin/trino --server localhost:8080
```

Try running a simple query:

```sql
SELECT 'Hello, Trino!' AS greeting;
```
